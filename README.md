INSTALLATION of git hub on vs code.

- Git is the backbone for connecting VS Code with GitHub.
- Download & install Git from git-scm.com.
- During installation, allow Git to be added to your system PATH.
2. Verify Git Installation
Open your terminal (inside VS Code or system terminal) and run:
git --version
If it shows a version number, Git is installed correctly.
3. Install VS Code
4. Install GitHub Extension in VS Code
- Open VS Code → go to Extensions (left sidebar).
- Search for GitHub Pull Requests and Issues.
- Install it — this adds GitHub integration directly into VS Code.
